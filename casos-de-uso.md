## Descrição dos Casos de Uso

### CDU 01 Cadastro

Cadastro de usuário.

**Fluxo Principal**

O usuário poderá se cadastrar utilizando o email.


**Fluxo Alternativo A**

O usuário poderá se cadastrar utilizando o número de celular.


**Fluxo Alternativo B**

O usuário poderá se cadastrar utilizando nome de usuário.
